# package.json
`engines` 用于指定项目运行的`node`、`npm`、`pnpm`等版本，目的减少版本差异产生的开发问题
`private` 安全性设置，防止被当作包发布
`name` 项目的名称，也是将来发布包的名字
`type:module` 使用es6语法
`main` 指定将来发布包的入口
