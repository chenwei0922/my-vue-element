## API
### Attributes

属性名     | 说明 | 类型 | 默认值
----------| --- | ---- | ---
name      | 图标名称    | `string` | ''
color     | 图标颜色    | `string` | ''
size      | 图标大小    | `string` / `number` | `20px`
svg-class | 自定义类名   | `string`            | ''
